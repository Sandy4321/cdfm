## Install
```
$ pip install cdfm
```

## Development
```shell
# 1. install develop dependencies
$ pip install -e .[dev]

# 2. linting
$ pylint cdfm

# 3. type checking
$ mypy @mypy_check_files --config-file=mypy.ini

# 4. testing
$ pytest
```
